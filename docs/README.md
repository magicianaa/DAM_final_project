# Documentation Index

This directory keeps project documents separate from source code and generated data.

## Project Documents

- `project/plans/`: movie recommendation project plans, development plans, and formatted PDFs.
- `project/design/`: system design documents for this movie recommendation project.
- `project/course/`: course or assignment reference files related to this project.

## Other Documents

- `other/`: historical or unrelated reference documents. For example, the air-quality plan is an expired topic proposal and is not part of the final movie recommendation deliverable.
